Around the Web

3D Game of Life

Using the orginal rules of Joseph Conway's Game of Life, I developed an enviroment that can explore the mechanics of node birth and death in an extra dimension. Using slightly edited rules, I was able to create a few patterns, including a combination of blinker and a block.

Walleye Census Evaluation

Using a decade of fish censuses for populations on Lake Erie, I graphed the against the amount of fisherman, other fish, and water conditions to discover a trend that looks largely similar to Fisheries-Induced Evolution

Closing Stocks Prediction Model

For a Kaggle Competition, we built a machine learning model that used hyperparameter optimization and CatBoost to predict prices the market would close at, based on both data Kaggle gave us and columns that we dervived based on the information given.

Github for Jupiter Notebook

Neural Network Midi Genre Classifier

Using ~13,000 songs from the Million Midi Dataset, I attempted to find an accurate way using neural networks to predict genre. This was done through a standard neural network, a standard convolutional neural network, and with convolution using zero crossings edge detection.
